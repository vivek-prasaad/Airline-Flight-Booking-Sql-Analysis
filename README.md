## Project Overview
This project is a MySQL-based Airline Flight Booking System designed to analyze airline booking data and generate business insights. The project demonstrates database design, data analysis, and SQL querying skills relevant to Data Analyst roles.

The system simulates real-world airline operations including flight details, passenger bookings, ticket pricing, and travel routes.

---

## Problem Statement
The objective of this project is to:

- Design a relational database for an airline booking system
- Store flight, passenger, and booking information
- Perform SQL-based data analysis
- Extract business insights such as revenue trends, booking patterns, and route performance

---

## Dataset Description
The dataset (`Airline.csv`) contains airline booking and flight-related data such as:

- Flight ID
- Airline Name
- Source & Destination
- Departure & Arrival Time
- Ticket Price
- Passenger Information
- Booking Details

The dataset was imported into MySQL for analysis.

---

## Database Design
The database includes structured tables such as:

- Flights
- Passengers
- Bookings
- Airlines

Primary and Foreign Keys are used to maintain relationships between tables.
